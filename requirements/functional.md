# Functional Requirements
## Browse meals
As a user, I want to browse the meals offered by the catering service so that I can learn which options are available to order.
## View ingredients of a meal
As a user, I want to view the list of ingredients required for each meal so that I know what ingredients will be ordered.
## Search meals
As a user, I want to search meals by name, ingredient, or category (e.g., vegan, gluten-free) so that I can quickly find meals suitable for my needs.
## Order ingredients for a meal
As a user, I want to order ingredients for a selected meal so that I can place my order automatically without manually selecting individual ingredients.
## Schedule delivery
As a user, I want to schedule ingredient delivery for the same day or the next day so that the delivery fits my time constraints.
## Track order status
As a user, I want to track the status of my ingredient orders so that I know whether they are pending, shipped, or delivered.
